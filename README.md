# 좌우충돌 좌충우돌 apple 사이트 clone 하기 프로젝트 🤪

<br>

저희 프로젝트는 **한 입 베어 문 사과 🍎 로고** 와 **"Think different"** 라는 슬로건으로 유명한 애플 기업의 공식 사이트 중 애플 고객지원 페이지를 클론하고 더 나아가 자체적으로 기존의 사이트에는 없던 기능을 추가해보는 프로젝트입니다.
